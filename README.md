# sanity-plugin-groq-list

Take advantage of GROQ to show matching documents for quick edit

## Installation

```
sanity install groq-list
```

## Configuration

The plugin can be configured through `<your-studio-folder>/config/groq-list.json`:

```json
{
  "title": "TAB title"
}
```
